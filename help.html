Trello Projects Chrome Extension Help

Trello Projects is a Chrome Extension that adds a little bit of functionality to Trello.com to make it perfect to use with multiple projects. Trello Projects allows you to add project labels to cards.

Adding project labels to a card

Project names are written in between parentheses in the title of a card. {Project} Upon an error I want to see an awesome 404 page. Upon saving the card, Trello Projects picks up the assigned project name and shows it as a badge in the lower left corner of the card.
